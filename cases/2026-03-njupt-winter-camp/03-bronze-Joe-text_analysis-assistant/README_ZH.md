# 基于AMD ROCm的通用文本深度分析助手
[English Version](./README.md)

## 项目简介
本项目是面向2026南邮寒假大作战AMD ROCm专题打造的高性能文本智能分析工具，专为aup-learning-cloud平台的Basic GPU Environment深度优化与适配。依托Jupyter Notebook交互式开发环境，项目实现了开箱即用、零配置部署的极简使用体验，同时深度结合AMD ROCm GPU硬件加速能力，将文本分析整体效率提升3-5倍，有效解决长文本分析卡顿、多文本对比流程繁琐、分析结果非结构化、上下文截断等实际工程痛点。

项目构建了覆盖单文本解析、多文本对比、超长文本处理、结构化报告导出的完整工作流体系：单文本支持摘要提炼、关键词提取、情感倾向判断、主题识别、核心论点归纳、逻辑结构拆解、论据提取、潜在假设挖掘、应用场景分析、局限性评估十大维度结构化解析；多文本可自动完成共性挖掘、差异对比、逻辑关联分析、互补价值评估，并生成标准化对比表格与分析报告；超长文本采用语义切分、滑动窗口、分块汇总三级处理策略，突破模型上下文长度限制，实现万字级文本完整无断层分析；最终支持Markdown与JSON双格式报告导出，兼顾可读性与二次开发能力，满足学习、科研、工程落地等多场景需求。

技术架构层面，项目采用全异步高可用设计，基于aiohttp实现异步API调用，搭配自动重试、限流保护、结果缓存等工程化机制，保障多用户并发场景下的稳定运行；通过思维链提示词工程与双语Few-shot示例优化模型输出质量，提升分析结果精准度与结构化程度；硬件层面轻量化适配办公级AMD GPU，无需高端显卡即可启用ROCm加速，在性能与硬件门槛间实现最佳平衡，既适合初学者快速上手实践，也可作为文本分析与ROCm加速开发的学习范例。

## 快速使用指南
1. 登录aup-learning-cloud平台，选择Basic GPU Environment环境并启动；
2. 进入项目根目录，执行依赖安装命令完成环境配置；
3. 在JupyterLab中打开main_zh.ipynb（中文版）或main.ipynb（英文版）；
4. 逐行运行所有代码单元格，等待环境检测与UI界面加载完成；
5. 通过交互式面板上传TXT格式文本文件，选择对应分析功能；
6. 等待分析完成后，查看结构化结果并导出Markdown/JSON格式报告。

## 核心功能模块
### 1. ROCm环境一站式检测
自动完成GPU硬件识别、ROCm加速状态校验、Ollama服务连通性测试、模型可用性检查、网络状态验证，输出可视化检测报告，对异常状态提供修复指引，确保工具运行环境完备。

### 2. 单文本十维度深度解析
- 文本摘要：生成50-100字客观精炼核心摘要；
- 关键词提取：识别10-15个高价值核心关键词与专业术语；
- 情感倾向：判断文本情感类型并输出置信度；
- 主题识别：精准定位文本核心主题与细分领域；
- 核心论点：归纳文本核心观点与主张；
- 逻辑结构：拆解文本段落关系与论证框架；
- 核心论据：提取支撑论点的数据、案例与事实依据；
- 潜在假设：挖掘文本未明确表述的前提与隐含假设；
- 应用场景：分析文本适用领域、使用边界与落地价值；
- 局限性：客观评估文本缺陷、偏差与可优化方向。

### 3. 多文本深度对比分析
支持2-5篇文本批量上传与并行分析，自动完成：
- 共性特征提取：识别文本共同主题、相似论点与一致立场；
- 关键差异对比：生成多维度对比表格，清晰呈现内容差异；
- 逻辑关联分析：判断文本间互补、递进、冲突、支撑等关系；
- 综合应用建议：给出多文本联合使用方案与价值整合策略。

### 4. 超长文本智能处理引擎
针对万字级以上长文本，采用三段式处理流程：
1. 基于语义与Token长度智能切分；
2. 分块并行分析并保留上下文关联；
3. 滑动窗口合并与全局摘要汇总；
彻底解决上下文截断问题，实现完整连贯分析。

### 5. 双格式结构化报告导出
- Markdown格式：排版美观、层次清晰，适合阅读、存档与文档编写；
- JSON格式：结构标准化，便于程序解析、数据入库与二次开发；
导出内容完整覆盖所有分析维度，无需手动整理。

### 6. 零代码交互式操作界面
全网页可视化操作面板，所有功能通过点击完成，支持文件上传、任务启停、进度展示、结果预览、历史记录、报告导出，完全屏蔽命令行操作，大幅降低使用门槛。

## 技术创新与亮点
1. **ROCm轻量化加速适配**
专为普通办公级AMD GPU优化，无需高端硬件即可启用GPU加速，分析效率提升3-5倍，实现性能与普及性的平衡。

2. **长文本智能处理算法**
创新语义切分+滑动窗口+分块汇总机制，完美支持32k Token超长文本，彻底解决传统工具分析断层、语义丢失问题。

3. **全异步高可用架构**
基于异步IO设计，搭配自动重试、限流控制、超时保护、结果缓存，稳定支持多用户并发使用，避免服务崩溃与请求阻塞。

4. **提示词工程深度优化**
内置思维链引导逻辑、结构化输出约束、双语Few-shot示例库，显著提升分析精准度、逻辑性与格式规范性。

5. **全链路工程化保障**
覆盖输入校验、敏感信息过滤、异常捕获、日志记录、错误重试，提升工具稳定性与健壮性。

6. **单文件开箱即用设计**
全部逻辑内嵌于Notebook文件，无复杂目录结构，部署极简、演示便捷、易于学习与复用。
